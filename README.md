# Netflix Analytics Dashboard

## Overview
Interactive Power BI dashboard analysing Netflix content trends, 
viewership patterns, and regional distribution.

## Key Insights
- Netflix content additions peaked between 2018–2020 with highest volume of originals
- Dramas and Comedies are the most produced genres globally
- USA, India, and UK are the top 3 content-producing countries
- Movies outnumber TV Shows at approximately 70% of total titles
- Ratings analysis shows TV-MA dominates the content library

## KPIs Built
- Total Titles, Movies vs Shows ratio
- Top Genres by count
- Content added YoY (time intelligence)
- Regional content distribution

## Tools Used
Power BI | DAX | Power Query | Data Modelling
